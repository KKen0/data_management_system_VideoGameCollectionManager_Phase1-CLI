# data_management_system_VideoGameCollectionManager (Phase 1)

A Java-based console application that manages a personal video game collection using object-oriented design and text-file data storage. Developed for Software Development I

## Features
- Add new video games
- Remove games from the collection
- Update hours played
- Display all games
- Generate backlog report
- Load and save game data using a text file

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- File I/O (Text File Repository)

## Project Structure
Game.java – represents a video game object  
GameManager.java – handles game management and file operations  
VGMApp.java – user interface and main application logic
